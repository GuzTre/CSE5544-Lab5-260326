How to run:

1) Unzip "Lab5_guzmantrevino1_AI" in a single repository
2) Open Repository Location (Should contain main files: "index.html", "README.txt")
3) Open "index.html" from Local Server Host (if in VS Code, install "Live Server by Ritwick Dey" extension, and from EXPLORER right click on "index.html", then click on "Open with Live Server").

You should be able to see the webpage containing the Final Scatterplot Chart for the LargeCSV, made with AI in D3 in localhost.
